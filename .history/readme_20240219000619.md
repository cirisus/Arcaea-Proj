# Arcaea Proj

This project aims to recreate the in-game scenes of Arcaea using front-end technologies.

## Features

- Reproduction of the gameplay user interface;
- Recreation of the visual effects
- Integration with online leaderboards

## Technologies Used

- HTML5
- CSS3
